  БЫСТРАЯ ИНСТРУКЦИЯ: Как сохранить данные после очистки кэша
═══════════════════════════════════════════════════════════════

🎯 САМЫЙ ПРОСТОЙ СПОСОБ (рекомендуется):

1. Откройте приложение (index.html)
2. Нажмите ПУСК → "Выбрать папку автосохранения"
3. Выберите эту папку (SimpleAIAdminka)
4. Готово! Все изменения теперь автоматически сохраняются в data.json

───────────────────────────────────────────────────────────────

⚡ ЧТО ЭТО ДАЕТ:

✅ Данные сохраняются АВТОМАТИЧЕСКИ в папку проекта
✅ НЕ НУЖНО копировать файлы вручную
✅ После очистки кэша данные ВОССТАНАВЛИВАЮТСЯ автоматически
✅ Файл data.json всегда актуальный

───────────────────────────────────────────────────────────────

📋 ЧТО ДЕЛАТЬ ПОСЛЕ ОЧИСТКИ КЭША:

1. Откройте приложение → данные загрузятся из data.json
2. Пуск → "Выбрать папку автосохранения" → выберите SimpleAIAdminka
3. Продолжайте работу!

───────────────────────────────────────────────────────────────

💡 ВАЖНО:

• Это нужно настроить ОДИН РАЗ
• Работает в Chrome 86+ и Edge 86+
• После очистки кэша нужно ПОВТОРНО выбрать папку (2 клика)
• Сами данные НЕ ТЕРЯЮТСЯ, теряется только настройка папки

───────────────────────────────────────────────────────────────

📖 Подробная инструкция: см. ИНСТРУКЦИЯ_ПО_СОХРАНЕНИЮ_ДАННЫХ.md
